# CLAUDE.md

此文件为 Claude Code (claude.ai/code) 在本仓库中工作时提供指导。

## 仓库概述

这是一个个人自学笔记仓库（刀仔的自学笔记），包含中文 Markdown 文档和各类编程主题的代码示例。

## 目录结构

仓库按编号的主题目录组织：

- `1. 算法/` - JavaScript 算法实现，详细解释在 `notes.md` 中
- `2. Vue篇/` - Vue.js 笔记，源码示例在 `源码集合/` 中
- `3. React篇/` - React 笔记
- `4. JavaScript篇/` - JavaScript 基础知识
- `5. TypeScript篇/` - TypeScript 笔记
- `6. LogicFlow篇/` - LogicFlow 库笔记
- `7. 工程化篇/` - 软件工程/工具笔记
- `8. 面试篇/` - 面试准备题目
- `9. NextJS篇/` - Next.js 教程章节（多个章节 .md 文件）
- `10. SQL篇/` - SQL 笔记

## 内容格式

- 大多数主题有一个主 `notes.md` 文件，包含结构化内容
- 算法目录包含按技巧分类的 JavaScript 实现文件（哈希、双指针、滑动窗口等）
- NextJS 目录有按编号的章节文件（第一章、第二章等）
- Vue 目录有 TypeScript/JavaScript 源码示例

## 笔记风格

- 所有笔记均使用中文编写
- 代码示例包含详细注释，说明时间/空间复杂度
- 算法笔记遵循 LeetCode 题目格式，包含题目描述、解法和解释
- NextJS 章节为教程风格，有逐步讲解
